# Projeto Órbita

## Como executar o programa

### Pré-requisitos
- MSYS64 instalado no sistema
- Compilador C++ (gcc/g++)

### Passos para execução

1. Abra o terminal MSYS64

2. No terminal MSYS64, navegue até o diretório do projeto:
```bash
cd "/c/Users/Tayler Tempest/Documents/Development/UFC/Computacao Grafica"
```

4. Rode o projeto com o comando:
```bash
bash run_orbit.sh
```

## Descrição

Projeto de simulação de órbitas planetárias desenvolvido em C++.
